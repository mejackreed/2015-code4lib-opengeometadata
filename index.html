OpenGeoMetadata @mejackreed
Geospatial data has a problem
Nobody* creates metadata
Nobody* shares metadata
There is software out there for sharing this metadata
But its yet another service that an institution has to run/maintain
Its not collaborative
It can be hard to use
Solution: OpenGeoMetadata
Uses Github as a platform
People and machines like Github
Collaborative way to share, harvest, and understand geospatial metdata
Extendable schema and standard agnostic
UVA, Princeton , Stanford, Tufts, Harvard
~ 50,000 metadata records shared so far
Shared toolkits forming in python and ruby
